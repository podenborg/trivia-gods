# Trivia Gods
A simple React Native trivia app that wraps the [Open Trivia API](https://opentdb.com/api_config.php).

## Tech Used
- TypeScript
- React
- React Native
- React Native Web
- Node.js / Express
- PostgreSQL

## Installation
### Front End
```
npm install --global expo-cli
cd app/
expo install
expo start
```

### Back End
```
cd api/
npm install
npm run dev
```
